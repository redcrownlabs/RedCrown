This release keeps the active catalog browse context intact while inspecting a
title.

## Fixes

- Back from movie or series details now restores the previous search term,
  category, genre filter, and sort order.
- Already loaded catalog pages and results remain available instead of
  restarting at page one.
- The catalog returns to the previous scroll position.
- Returning from details no longer issues an unnecessary initial catalog
  request when the retained results still match the active query.

## Upgrade notes

- This update does not change library data, settings, or cache formats.
- Windows builds remain unsigned pre-releases. Verify downloads with the
  attached `SHA256SUMS.txt` file.
